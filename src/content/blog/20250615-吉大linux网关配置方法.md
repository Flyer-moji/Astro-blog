# 吉大linux网关配置方法

随便打开一个浏览器网页，在**连接校园网后**都会自动跳转到吉大校园网的登录页面。

如果无法自动跳转到登录页，可尝试访问：http://login.jlu.edu.cn

寻找到linux的下载地址，选择对应的版本，下载完成后双击解压，进入到解压后的文件夹中。

在DrClient目录下右键调出终端，输入以下命令：

```
sudo ./privillege.sh
./DrClientLinux
```

程序运行后输入 校园网账号和密码即可成功上网
