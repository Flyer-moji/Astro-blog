## DML (Data Manipulation Language)
对表中数据进行操作。包括插入、删除、更新等。

### 插入

`insert`

添加单组数据：
```SQL
insert into table_name (column1, column2, column3,...) values (value1, value2, value3,...) #指定字段赋值

insert into table_name values (value1, value2, value3,...) #全部赋值
```
批量添加数据：

```SQL
insert into table_name (column1, column2, column3,...) values  #指定字段赋值
(value1, value2, value3,...),
(value1, value2, value3,...),
(value1, value2, value3,...)；

insert into table_name values  #全部赋值
(value1, value2, value3,...),
(value1, value2, value3,...),
(value1, value2, value3,...)；
```


### 修改

`update`

```sql
update table_name set column1=value1, column2=value2, column3=value3 [where condition];
```

不写where，则全部更新。

### 删除

`delete`

```sql
delete from table_name [where condition];
```
