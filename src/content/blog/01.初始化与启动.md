## 启动
在安装时可以设置mysql为windows服务，也可以手动启动mysql服务。

启动方法：

管理员权限打开cmd

`net start mysql80`
`net stop mysql80`

或者手动打开：cmd -> 输入`services.msc` -> 找到mysql80 -> 右键启动/停止

## 进入客户端

可视化操作：在自己的程序中找到mysql command client,输入密码即可。当前设置密码为：`python`

cmd命令行操作：

`mysql [-h host] [-p port] -u root -p`

输入密码，如：`python`

注意:

1.[]内为可选参数（host为服务器地址，port为服务器端口），默认时不用填写。

2.这种方法需要配置系统的环境变量。
